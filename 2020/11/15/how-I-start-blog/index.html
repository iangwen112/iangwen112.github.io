如何架設Blog

2020-11-15
技術相關

使用工具

我使用以下工具完成架設Blog：

Hexo - 一個免費Blog架構

Ocean - 一種套用在Hexo上的主題(還有其他主題請參考Hexo網站)

GitHub/GitHub Pages - 存放及部署Blog程式、文章、圖片等資料的地方

Visual Studio Code - 用來設定程式、寫作，支援markdown preview

Typora - 專門寫markdown文件的編輯器
